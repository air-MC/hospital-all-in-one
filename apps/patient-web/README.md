# Admin Web (React)

Part of the Hospital All-in-One Platform (Track C).

## Setup
1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## Features
- **Slot Manager**: View and generate slots for the backend.
- Connects to Backend API at `http://localhost:3000`.

## Directory Structure
- `src/hooks/useSlotManager.ts`: React Query/SWR hooks for API.
- `src/App.tsx`: Main Dashboard UI.
