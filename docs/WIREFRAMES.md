# 3. 핵심 화면 와이어프레임 (Wireframes)

모바일 앱은 **Flutter**, 관리자 웹은 **React**로 개발됩니다.

## A. 환자용 앱 (Mobile - Flutter)

### 1. 예약 슬롯 선택 화면 (Booking Slot)
직관적인 10분 단위 그리드와 마감 임박 표시가 핵심입니다.

```
+-----------------------------------+
|  <      2026년 1월 17일      >   |
| [월] [화] [수] (목) [금] [토] [일]|  <-- 가로 스크롤 날짜 선택
+-----------------------------------+
| 오전 진료                         |
|                                   |
| [ 09:00 ]  [ 09:10 ]  [ 09:20 ]   |
| (2/3명)     (마감)      (1/3명)    |
|  OPEN        FULL       OPEN      |
|                                   |
| [ 09:30 ]  [ 09:40 ]  [ 09:50 ]   |
|   ...         ...        ...      |
+-----------------------------------+
| [ 예약하기 버튼 (Selected: 09:20) ] | <-- 하단 고정 (Sticky)
+-----------------------------------+
```

### 2. 수술/입원 케어 홈 (Surgery Care Home)
수술 환자 전용 대시보드입니다.

```
+-----------------------------------+
|  홍길동님, 수술까지 D-2일 남음     |  <-- 강조된 헤더 (Teal Color)
|  (무릎 인공관절 수술)              |
+-----------------------------------+
| [ 오늘 해야 할 일 ]                |
|  □ [14:00] 항생제 반응 검사        |
|  □ [18:00] 금식 시작 (물 포함)     |
+-----------------------------------+
| [ 바로가기 메뉴 ]                  |
|  [체크리스트]  [식단안내]          |
|  [복약알림]    [검사일정]          |
+-----------------------------------+
```

## B. 병원 관리자 (Web - React)

### 1. 과별 운영시간 설정 (Schedule Settings)

```
+-------------------------------------------------------+
| 과 선택: [ 정형외과 v ]                                  |
+-------------------------------------------------------+
| 요일별 설정                                            |
| [v] 월요일 | 09:00 ~ 18:00 | 점심: 12:00~13:00 | [저장] |
| [v] 화요일 | 09:00 ~ 18:00 | 점심: 12:00~13:00 | [저장] |
| [ ] 수요일 | (휴진)        |                  |       |
+-------------------------------------------------------+
| 자동 생성 옵션                                         |
| - 슬롯 단위: [ 10분 ]                                  |
| - 슬롯당 정원: [ 3명 ]                                 |
| [ 슬롯 생성하기 (Batch Run) ]                          |
+-------------------------------------------------------+
```

### 2. 케어 플랜 관리 (Care Plan Manager)

```
+-------------------------------------------------------+
| 환자명: 홍길동 (수술일: 2026-01-20)                      |
+-------------------------------------------------------+
| [ + 항목 추가 ]                                        |
+-------------------------------------------------------+
| 타임라인                                               |
| [01-19 14:00] [검사] MRI 촬영 (본관 2층)          [수정]|
| [01-19 18:00] [식사] 연식 (죽)                   [수정]|
| [01-20 08:00] [수술] 수술실 이동                  [수정]|
+-------------------------------------------------------+
```
