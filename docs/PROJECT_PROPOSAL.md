| 단계 | 기간 | 주요 과제 | 산출물 |
|:---:|:---:|:---|:---|

## B. 개발 견적 (Resource-based Estimate)

본 견적은 고정 비용이 아닌 투입 인력(Man-Month) 기반의 표준 단가 예시입니다.

### 1. 투입 인력 구성
- **Project Manager (PM/PL)**: 0.5 M/M (전체 일정 관리 및 설계)
- **Senior Backend Dev**: 1.0 M/M (NestJS, DB, 예약 엔진 설계)
- **Fluid App Dev**: 1.0 M/M (Flutter iOS/Android)
- **Frontend Dev**: 0.5 M/M (React Admin)

### 2. 예상 개발 비용 (Total)
*산정 기준: 중급 이상 개발자 프리랜서 단가 기준 (시장 평균)*
- **총 예상 비용**: **약 5,000만 원 ~ 7,000만 원** (VAT 별도)
- **상세 내역**:
  - 백엔드/아키텍처: 2,500만 원
  - 모바일 앱 개발: 2,500만 원
  - 관리자 웹/디자인: 1,500만 원
